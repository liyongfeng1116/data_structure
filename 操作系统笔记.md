# Operating Systems
## 操作系统定义
- 操作系统是一个控制软件（用户）
- 资源管理，分配资源，效率最大化（内存、cpu、硬盘）
- 硬件之上，应用程序之下
- 分为外壳shell（linux，windows，android）内核kernel是操作系统的重点
## 操作系统内部组件
1. cpu调度
2. 内存管理（物理内存，虚拟和内存）
3. 终端管理
4. 磁盘管理
5. 文件管理
## 操作系统特征：
- 并发（一段时间） 并行（一个时间点，一般多个cpu）
- 共享
- 虚拟
- 异步
## 计算机工作原理
代码放到内存中，控制器自动取地址进行执行（取址执行）
BIOS base input output system
X86 intel架构
